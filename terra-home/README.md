# 🏠 Property Listing Website

This is a multi-user property listing website designed for both **clients** and **real estate agents**.  

The website allows clients to browse properties and agents to register, log in, and manage property listings.

---

# 📌 Project Description

This project was built using:

- HTML
- CSS
- JavaScript
- LocalStorage (for login system)

It simulates a real estate platform where:

- Clients can create accounts and log in
- Agents can register and log in
- Agents can access a dashboard


---

# 👥 User Roles

## Client

Clients can:

- Create an account
- Log in
- Browse available properties
- View property details

---

## Agent

Agents can:

- Register as verified agents
- Log in securely
- Access agent dashboard
